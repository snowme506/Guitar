# Guitar - 吉他学习网站（儿童版）

## 1. Concept & Vision

一个专为 **4-12岁儿童** 设计的吉他学习乐园 🎸✨

核心理念：**像玩游戏一样学吉他**。没有枯燥的说教，只有：
- 🎮 打怪升级式的课程进度
- ⭐ 收集星星换奖励
- 🎉 每一点进步都有欢呼声
- 🤝 一个永远不会批评你的"吉他小伙伴"

吉祥物：**弹弹** — 一只热情活泼的小吉他，会在孩子弹得好时跳舞，弹得不太顺时喊"加油！你可以的！"

## 2. Design Language

### Aesthetic Direction
**卡通游乐园 + 糖果色系 + 圆润Q弹**

参考：任天堂游戏的欢快感 + 可爱卡通风格 + 无障碍儿童界面

### Typography

```css
/* 中文使用圆润友好的字体 */
--font-heading: 'ZCOOL KuaiLe', 'Nunito', cursive;  /* 标题：童趣圆润 */
--font-body: 'ZCOOL XiaoWei', 'Nunito', sans-serif; /* 正文：清晰圆润 */

/* 英文辅助 */
--font-english: 'Nunito', 'Comic Neue', cursive;
```

**字号规则（放大30% vs 成人版）：**
- 主标题: 32px (最小)
- 副标题: 24px
- 正文: 20px (最小)
- 辅助: 18px
- 谱子音符: 28px (保证可读性)

### Spatial System

- **基础单位**: 12px (比成人版更大)
- **按钮最小尺寸**: 64px × 64px (方便小手点击)
- **圆角**: 全部使用大圆角 16px-24px
- **卡片阴影**: 柔和彩色阴影 (带颜色倾向，比黑灰更有趣)
- **间距**: 松散，元素之间留足空间，不拥挤

### Motion Philosophy

**核心原则：每一个动作都要有回应！**

| 动作 | 动画 | 时长 |
|------|------|------|
| 点击按钮 | 弹跳缩放 (scale 0.95→1.1→1) | 300ms |
| 答对/成功 | 星星爆炸 + 弹跳庆祝 | 600ms |
| 录音中 | 音符跳动 + 圆圈呼吸 | 持续 |
| 得分展示 | 数字滚动 + 撒花效果 | 1200ms |
| 页面进入 | 元素依次弹入 (像游乐园开幕) | 400ms |
| 错误 | 摇晃 + 鼓励文字 | 500ms |
| 升级/成就 | 全屏撒花 + 音效 + 弹弹跳舞 | 2000ms |

**弹弹的动画状态：**
- 待机：微微上下浮动，2s循环
- 开心：原地转圈 + 星星围绕
- 加油：双手举起摇动 + "你可以的！"气泡
- 惊喜：眼睛变大 + 弹跳

### Visual Assets

**图标风格：**
- 全部使用填充型图标（实心，不要线稿）
- 带圆角、柔和边缘
- 颜色鲜亮，不超过2色

**吉祥物弹弹：**
```
    🎸
   ╭───╮
   │ ◕‿◕ │   ← 可爱吉他小人，圆圆的眼睛
```

**装饰元素：**
- 音符、小星星、彩虹条纹
- 云朵、太阳、花朵（自然元素）
- 进度条充满流动感的彩虹渐变

## 4. Features & Interactions

### 4.1 课程体系（游戏化）

**管理员上传功能（家长/老师）：**

支持从手机、iPad、电脑直接上传课程内容：

| 上传类型 | 支持格式 | 用途 |
|---------|---------|------|
| 谱子图片 | JPG、PNG、PDF | 跟弹谱子 |
| 和弦图 | JPG、PNG | 和弦示意 |
| MusicXML | .xml | 可交互谱子（高级） |

上传流程：
```
[+] 添加新课程
    │
    ├── 🎼 上传谱子图片  →  拍照 或 从相册选择
    │                      (拍琴谱/黑板/书本)
    │
    ├── 🔍 网络搜索谱子  →  WebView 内嵌搜索网站
    │                      (Ultimate Guitar / Chordify)
    │
    ├── 🎸 上传和弦图   →  拍照 或 从相册选择
    │
    └── ⭐ 设置课程信息
            - 课程名称
            - 难度星级
            - 适合年龄
```

**上传组件特性：**
- 📱 **移动端优先**：大按钮、一键拍照/选相册
- 📷 支持拍照上传谱子（现场拍，现场学）
- 🗂 批量上传：一次选多张图片
- ☁️ 上传到云存储（后期）或本地base64（快速原型）
- 📊 上传进度条 + 预览

**数据结构：**
```typescript
interface Course {
  id: string;
  title: string;           // "认识吉他"
  emoji: string;           // 🎸
  coverColor: string;      // 卡片背景色
  coverImage?: string;     // 课程封面（可上传）
  lessons: Lesson[];
}

interface Lesson {
  id: string;
  title: string;           // "第1课：拨弦练习"
  duration: number;        // 秒（自动识别）
  starsToUnlock: number;    // 需要多少星星解锁
  content: {
    video?: { 
      url: string;         // 上传后生成的URL
      duration?: number;   // 自动提取
    };
    sheet?: string;        // 谱子图片URL
    chordDiagram?: string; // 和弦图URL
    musicXml?: string;     // MusicXML（可选）
  };
  standardNotes?: Note[];   // 用于评分（可选）
}
```

**游戏化元素：**

| 元素 | 说明 |
|------|------|
| ⭐ 星星 | 每课练习获得，1-3颗（根据表现） |
| 🏆 徽章 | 收集一定星星解锁特殊成就 |
| 📊 进度条 | 显示今日目标/本周目标 |
| 🎁 奖励 | 连续练习3天可获得装饰（弹弹的新衣服） |
| 🔓 解锁 | 星星够了自动解锁新课程 |

**解锁逻辑：**
- 第1课：免费
- 第2课：需要5颗星
- 第3课：需要10颗星
- ...

### 4.2 示范视频

> ⚠️ **改动：已移除演示视频功能。** 课程以谱子跟弹为核心，简化界面。

### 4.3 谱子跟弹（上传 + 网络搜索双轨）

**谱子来源有两种方式：**

**方式一：上传谱子图片（保留）**
- 拍照或从相册选择谱子图片
- 支持 JPG、PNG、PDF
- 上传后即时预览

**方式二：网络搜索（新增 WebView）**
- 在课程编辑页点击 🔍 "网络搜索" 按钮
- 弹出 WebView 全屏窗口
- 内嵌吉他谱网站：
  - Ultimate Guitar Tabs (https://ultimate-guitar.com)
  - Chordify (https://chordify.net)
  - Guitar.com (https://guitar.com)
- 搜索到想要的谱子后：
  - 复制谱子图片 URL
  - 粘贴到课程编辑的"谱子链接"输入框
  - App 加载显示

**设计要点：**
- 谱子图片要**大、要清晰**
- 音符标注**中文**："空弦"、"1品"、"2品"
- 当前音符用**绿色圆圈**高亮（绿色=正确！）
- 点击音符，播放该音的吉他音效

**简化控制条：**
```
╭─────────────────────────────────────────────╮
│  [◀◀] [▶] [▶▶]     🔁 循环    [1x] [2x]    │
╰─────────────────────────────────────────────╯
```

**自动滚动**：视频播放时，谱子跟着滚，滚到哪亮到哪

### 4.4 老师点评打分（录音 only）

**核心理念：真人对弹奏进行点评，更有温度、更准确。**

> ⚠️ **改动：录像功能已移除，只保留录音。** 减少权限请求（不需要相机权限），界面更简洁，存储体积更小。

**流程：**
```
学生录音
    ↓
上传录音到"待点评"队列
    ↓
老师（家长）听到录音
    ↓
老师选择评分 + 填写点评
    ↓
学生收到点评 + 星星奖励
```

**家长/老师端（Admin）：**

| 功能 | 说明 |
|------|------|
| 📋 待点评列表 | 显示所有等待点评的录音 |
| 🎧 一键播放 | 点击播放学生录音 |
| ⭐ 打分按钮 | [1星] [2星] [3星] [4星] [5星] |
| 💬 点评输入 | 文本框，老师输入鼓励的话 |
| ✅ 提交点评 | 发送给学生 |

**学生端（收到点评）：**

```

║                                                ║
║         🌟🌟🌟🌟🌟  5星！太棒了！！            ║
```

**点评话术（老师端引导）：**

预设鼓励模板，老师可选择或自定义：
- "节奏很稳，继续加油！"

## 6. Technical Approach

### 技术栈
- **Framework**: React 18 + Vite
- **Styling**: Tailwind CSS + CSS Variables
- **动画**: Framer Motion (儿童产品需要丰富动效)
- **谱子**: 预制图片（儿童版不需要VexFlow，降低复杂度）
- **音频**: MediaRecorder API + pitchfinder
- **状态**: Zustand (轻量)
- **音效**: Howler.js (音频管理)

### 文件上传方案
- **react-dropzone**: 拖拽 + 点击上传
- **相机/相册**: react-image-picker (React Native风味的Web方案)
  - `input type="file" capture="environment"` 调用摄像头
  - `input type="file" accept="video/*"` 选择视频
- **移动端兼容**: 使用 `<input type="file">` 的 `capture` 属性直调相机
- **预览**: 上传前即时预览（图片、视频缩略图）
- **存储**: 
  - 快速原型：localStorage + base64 / Blob URL
  - 正式版：上传到云存储（OSS/COS/S3）

### 项目结构

### 数据模型

```typescript
interface Progress {
  totalStars: number;
  earnedBadges: string[];
  outfits: string[];              // 解锁的弹弹装饰
  lessons: {
    [lessonId: string]: {
      completed: boolean;
      bestScore: number;
      starsEarned: number;
      attempts: number;
      lastPlayed: string;
    };
  };
}

interface Reward {
  id: string;
  name: string;
  cost: number;
  type: 'outfit' | 'badge' | 'effect';
  emoji: string;
}
```

### 儿童产品特殊考虑

| 方面 | 方案 |
|------|------|
| 麦克风权限 | 首次引导用卡通教程解释为什么要授权 |
| 录音时长 | 最长60秒（防止儿童困惑卡住） |
| 错误提示 | 永远用弹弹的鼓励话术，不用技术术语 |
| 退出确认 | 任何离开页面的操作都要确认 |
| 家长模式 | 双击头像3次进入，可查看详细数据 |
